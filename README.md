# Silus Pronunciation
How to say "Silus" in Latin. The correct way.

1. Classical Latin (SIH-loos)
This is the pronunciation used in Ancient Rome (roughly 1st century BCE to 2nd century CE): 

Si: Pronounced with a short "i" sound, like the "si" in "sit".<br>
lus: Pronounced with a short, rounded "u," similar to the "u" in "put".<br>
Phonetic (IPA): [ˈsɪ.ɫʊs]. <br>
